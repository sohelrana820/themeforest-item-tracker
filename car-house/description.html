Car House is a car dealer template. Which contains 10 individual files with 100% responsive & W3C HTML validate coding. This template has been used one color and it has two different layout for car listing. All the HTML& CSS code are commented properly so the developer can easily customize the template whatever they needed.

Core Features:

  • 100% Responsive Layout
  • 100% W3C valid code
  • 10 HTML file
  • Carousel Slider
  • Simple Dropdown Navigation
  • List / Grid view on car list
  • Google Fonts used
  • Font Awesome Icons
  • Using Twitter Bootstrap (version 3.3.6)
  • Easy to Customize.
  • Properly commented in HTML & CSS codes.
